Analyse der Nutzung von KI für AENEAS (ANuKI) - Repository

This repository contains source code of software projects related to ANuKI. Further information about the project, its contributors and funding can be found on the website of Reutlingen University or on the website of the Hermann Hollerith Zentrum.

If you are interested in a commercial use of the ANuKI results, please contact the German Space Agency directly. Use Dr. Carsten Bauer as contact person.
